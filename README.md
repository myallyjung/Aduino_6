## 프로젝트 소개
> 딥러닝 이미지 인식 기술(Teachable Machine)과 아두이노를 연동하여, 쓰레기의 종류를 스스로 판단하고 올바른 분리수거를 LED 색상으로 안내하는 지능형 분리수거 보조 로봇을 제작을 목적으로 합니다. 인공지능 이미지 인식 기술을 실생활에 직접 적용하여 올바른 분리수거를 유도하고, 최종적으로 자원 재활용률을 향상시키고자 합니다.

## 시스템 구조

> *에코 소터(Eco Sorter)는 사용자가 쓰레기를 제시하는 것부터 LED 알람이 켜지기까지 총 5가지 단계의 유기적인 프로세스를 동작합니다. *
```
[1단계: 영상 입력] ──> 사용자가 카메라(웹캠) 앞에 쓰레기 제시
        │
        ▼
[PC (AI 분석부)]
[2단계: AI 분석] ───> Teachable Machine (MobileNetV2 기반) 실시간 분석
        │
        ▼
[3단계: 재질 분류] ───> AI 모델이 [플라스틱 / 캔 / 종이] 판별
        │
        ▼
[4단계: 신호 전달] ───> PC에서 아두이노 보드로 시리얼 통신(USB) 데이터 전송
        │
        ▼
[Arduino Uno (하드웨어 제어부)]
[5단계: 하드웨어 제어] ─> 수신된 결과에 따라 LED 점등 (초록/파랑/노랑)
```

## 사용 기술 및 부품

### 소프트웨어
| 항목 | 내용 | 구분 |
|------|------|------|
| Vision AI | AI 학습 및 이미지 분류 | Teachable Machine (MobileNetV2 아키텍처 기반) |
| 개발 언어 (PC) | AI 제어 및 이미지 처리 언어 | Python 3.x |
| 개발 언어 (아두이노)| 하드웨어 펌웨어 제어 언어 | C++ (Arduino C/C++) |
| 통신 프로토콜 | 기기 간 데이터 전송 | 시리얼 통신 (Serial Communication over USB, 9600 Baud) |
| 개발 환경 (IDE) | 통합 프로그램 개발 환경 | VS Code / Arduino IDE 2.x |

### 하드웨어
| 부품 | 수량 | 용도 |
|------|------|------|
| Arduino Uno  | 1 | 결과 수신 및 출력 신호 제어를 담당하는 메인 마이크로컨트롤러 |
| PC 웹캠 | 1 | 실시간 영상 입력 및 분석용 프레임 획득을 담당하는 입력 장치 |
| LED (초록색) | 1 | 2번 핀에 연결. AI 판별 결과 플라스틱(PLASTIC, 'L') 배출 구분 가이드용 |
| LED (파란색) | 1 | 3번 핀에 연결. AI 판별 결과 캔(CAN, 'C') 배출 구분 가이드용 |
| LED (노란색) | 1 | 4번 핀에 연결. AI 판별 결과 종이(PAPER, 'P') 배출 구분 가이드용 |
| 브레드보드 | 1 | 회로 구성 |
| M/M 점퍼 와이어 | 4(긴 거 1개, 짧은 거 3개) | 아두이노의 디지털 제어 핀(2, 3, 4, GND)과 브레드보드를 연결하는 배선재 |
| 220옴 저항 (Resistor) | 3 | 각 LED에 가해지는 과전류를 차단하여 소자 파손을 방지하는 보호용 저항 |

## 구현 방법

### 1. Vision AI 모델

- **모델 선택 이유** : (실시간 추론 속도가 매우 빠르고 경량화가 용이하여 단말 환경에서도 지연 없이 부드럽게 작동하는 MobileNetV2 기반의 Teachable Machine 모델을 연동 모델로 선택)
- **데이터셋** : (학습에 필요한 플라스틱, 캔, 종이의 다양한 재질 및 형태별 쓰레기 이미지를 직접 촬영하여 사용하였으며, 웹 환경에 배포된 호스팅 API 주소(https://teachablemachine.withgoogle.com/models/YUx8KPgRr/)를 통해 메타데이터 라벨을 실시간으로 동기화합니다)
- **학습 방법** : (전이 학습(Transfer Learning) 방식을 적용하여 기학습된 특징 데이터를 기반으로 수집 데이터를 미세 조정 학습시켰습니다. 파이썬 스크립트 상에서 입력 프레임을 224x224 정밀 규격으로 전처리 및 정규화하고 특성값 매핑 알고리즘을 거쳐 실시간 분류를 수행합니다.)


### 2. 아두이노 제어 로직

- **수신된 명령어에 따라 LED 제어** : PC 측 파이썬 프로그램으로부터 USB 시리얼 통신(9600 Baud)을 통해 문자 신호 'L', 'C', 'P'를 전송받으면, 아두이노 우노의 디지털 출력 제어로 각각 2번 핀(초록 LED), 3번 핀(파란 LED), 4번 핀(노란 LED)을 3초간 HIGH 상태로 점등한 후 자동으로 LOW로 낮춥니다.
- **LED 신호로 시스템 상태 시각화** : 분류 판정을 알리는 특정 LED 소자가 켜질 때 다른 나머지 출력 핀들을 즉각 LOW로 차단 제어(인터록 기법)하여 다중 발광으로 인한 시스템 오인지 상태를 방지하고 직관적인 배출 위치를 밝혀 시각화합니다.

## 기대 효과 및 활용 방안
> 에코 소터(Eco Sorter) 시스템은 비전 AI 기술과 피지컬 컴퓨팅을 일상적인 분리배출 활동에 접목하여 다음과 같은 뛰어난 기대 효과와 사회적 가치를 가집니다. 

1. [기대 효과]
<실생활 적용을 통한 자원 재활용률 향상> : 인공지능 이미지 인식 기술을 실생활에 직접 적용하여 올바른 분리수거를 유도하고, 최종적으로 자원 재활용률을 향상시키는 데 뛰어난 기대효과를 지닙니다.
<시각적 가이드라인을 통한 환경 교육> : 시각적 피드백 시스템(LED 점등)을 통해 직관적인 배출 위치를 안내함으로써 환경 보호 및 분리배출 교육 효과를 동시에 거둘 수 있습니다.

2. [구체적인 활용 방안]
<교육 및 공공시설의 'AI 분리수거 로봇' 도입> : 이러한 기술적·사회적 가치를 바탕으로 학교, 구청, 유치원 등 교육 및 공공시설의 쓰레기 수거함에 이 시스템을 탑재하여 사용할 수 있습니다. 또, 분리배출이 서툰 아이들에게는 올바른 환경 습관을 길러주는 유용한 교구로 기능할 수 있으며, 시민들에게는 스마트 시티를 직접 체감할 수 있는 상징적인 서비스로 기능할 수 있습니다.
<다중이용시설 배치 및 상시 안내 시스템 운영> : 유동인구가 많고 분리배출이 잘 지켜지지 않는 대학 캠퍼스, 아파트 단지, 공원 등 다중이용시설에 본 로봇을 배치하여 정밀한 분리가 가능합니다. 그렇기에 야간이나 관리인이 없는 시간대에도 LED 가이드를 통해 시민들이 정확한 위치에 쓰레기를 버릴 수 있도록 상시 안내 시스템으로 작동시킬 수 있습니다.

## 향후 개선 사항
> 본 에코 소터 시스템의 인식 성능을 고도화하고 실생활에서의 실용성을 더욱 높이기 위해 다음과 같은 구체적인 학습 전략을 추진합니다.

1. 다양한 환경의 데이터를 활용한 데이터셋 확장
[환경 변수 반영] - 실제 다양한 수거 환경을 고려하여 크기, 색상, 형태뿐만 아니라 촬영 각도, 조명 세기 및 방향 등 조건이 서로 다른 여러 상황의 쓰레기 이미지를 추가 수집하여 학습 데이터셋을 확장합니다. 이를 통해 모델의 환경 적응력과 안정성을 극대화합니다.

2. 오분류 피드백 기반의 약점 보완 학습 (Feedback Loop)
[사례 분석 및 재학습] - 실제 구동 학습 과정에서 오분류된 구체적인 사례를 체계적으로 분석합니다. 예를 들어 '캔'을 '종이'로 잘못 인식한 경우, 해당 오분류 이미지를 새로운 재학습 데이터에 집중적으로 추가 구성합니다.
[점진적 성능 향상] - 이러한 피드백 기반 피쳐 보완 학습 방식을 통해 AI 모델의 약점을 메우며, 최종 적용 환경에서도 한층 더 안정적이고 정밀한 분류 정확도를 얻을 수 있도록 합니다.

3. 인식 가능한 재활용 품목 범위의 점진적 확대
[신규 품목 수집] - 초기 타겟인 플라스틱, 캔, 종이 분류를 넘어 실생활 배출 빈도가 매우 높은 비닐류, 유리병, 종이팩 등 다양한 재활용 품목군의 데이터를 추가 수집 및 학습합니다.
[통합 분리수거 솔루션 구축] - 점진적으로 인식 스펙트럼을 넓혀가며 한층 더 완전하고 실용적인 AI 기반 분리수거 토탈 안내 시스템으로 발전시킬 계획입니다.
